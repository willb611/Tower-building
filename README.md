## What is this repo?
Simulation written in Akka of 'Tower building', where builders place colored blocks on Towers. Log outputs of the 'winning' color at regular points in time.

## Requirements to run 
Need:
- sbt installed

### Actually running it
- Enter sbt prompt (i.e. type `sbt` in console)
- `run` To run it
- `test` to run tests
